## 4. 距离判定说明（当前实现）

- 先在“正对目标”阶段收集参考面积 `A_ref`（短窗）。
- 实时面积 `A` 经滤波得到稳定值。
- 用比例 `R = A / A_ref` 做等级判定（带进入/退出滞回）。
- 语音按等级升级触发，避免来回反复播报。
